# Welcome

Srotas Space Docs is a fast, static documentation site generated from Markdown.

## What you get

- Multi-language docs under `/en` and `/hi`
- Rust + Actix dev server
- Static HTML output for hosting
- Tailwind styling pipeline

## Next steps

1. Add more Markdown files under `docs/en`.
2. Run the build to generate `public/`.
3. Deploy the `public/` folder to your host.
